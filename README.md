# 🍔 Burger Builder App

A dynamic and interactive Burger Builder web application built using React. Users can customize their burger, add or remove ingredients, view pricing in real-time, and place orders.

---

## 🚀 Live Demo

👉 Add your Firebase link here:
https://my-burger-app-aabca.web.app/auth

---

## 🧠 Features

* Build your own burger with ingredients
* Add / remove ingredients dynamically
* Real-time price calculation
* Order summary modal
* Checkout process with contact form
* Authentication (Login / Signup)
* Error handling for API calls

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **State Management:** Redux
* **HTTP Requests:** Axios
* **Styling:** CSS Modules
* **Deployment:** Firebase Hosting
* **Version Control:** Git & GitHub

---

## 📂 Project Structure

```
src/
 ├── components/       # UI components (Burger, Controls, Navigation)
 ├── containers/       # Main logic containers (BurgerBuilder, Checkout)
 ├── hoc/              # Higher Order Components
 ├── store/            # Redux actions & reducers
 ├── assets/           # Images & static files
 └── shared/           # Utility functions
```

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
https://github.com/nagavenim8951-hash/Burger-app.git
```

2. Navigate to the project folder:

```bash
cd burger-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

---

## 🚀 Build & Deployment

### Build the project

```bash
npm run build
```

### Deploy using Firebase

```bash
firebase deploy
```
